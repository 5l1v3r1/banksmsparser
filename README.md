# Библиотека для распознавания смс от банков
Библиотека содержит набор шаблонов в формате xml и класс для работы с ними

Список банков:
* Сбербанк России
* Тинкофф
* Альфа-Банк
* ВТБ24 -> ВТБ
* QIWI wallet
* Газпромбанк
* ТЕХНОБАНК (Республика Беларусь)
* УБРиР (Уральский банк реконструкции и развития)
* Райффайзенбанк
* МТС-Банк
* UniCredit Bank
* Ощадбанк (Украина)
* KaspiBank (Казахстан)
